Distillery

Transform cluttered web articles into clean, focused reading experiences — then reads them to you. No ads. No tracking. No cloud.

Download for macOS View on GitHub
article

Clean Reading

Strips ads, popups, and clutter from any article. Multiple fallback sources for paywalled content.

auto_awesome

AI Summaries

Stream bullet-point summaries directly in your browser using your own OpenRouter API key.

headphones

Text to Speech

On-device neural TTS with Kokoro. Fully offline after initial model download. Natural-sounding audio.

speed

Speed Reader

RSVP-style rapid reading at 400+ WPM. Words displayed one at a time for focused consumption.

Running from source?
This app requires a Python backend. Clone the repo and run:
pip install -r requirements.txt && uvicorn main:app --port 8000
